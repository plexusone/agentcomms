# AgentComms Roadmap

This roadmap documents planned and completed features for this project.

The format is based on and generated by [Structured Roadmap](https://github.com/grokify/structured-roadmap), which provides a machine-readable JSON intermediate representation with deterministic Markdown generation.

## Overview

AgentComms is an MCP plugin that enables voice calls and chat messaging for AI coding assistants (Claude Code, AWS Kiro CLI, Gemini CLI), built with the plexusone stack (omnivoice, omnichat, mcpkit). Start a task, walk away. Your phone rings when the AI is done, stuck, or needs a decision—or get notified via Discord, Telegram, or WhatsApp.

## Technology Stack

- **omnivoice** - Voice abstraction layer (TTS, STT, Transport, CallSystem interfaces)
- **omnichat** - Chat abstraction layer (Discord, Telegram, WhatsApp)
- **omnivoice-twilio** - Twilio implementation of omnivoice interfaces
- **mcpkit** - MCP server runtime with built-in ngrok integration
- **assistantkit** - Multi-tool plugin configuration management (Claude, Kiro, Gemini)

## Multi-Tool Support

AgentComms supports multiple AI coding assistants through the assistantkit bundle API:

- **Claude Code** - Plugin manifest, skills, commands, hooks
- **AWS Kiro CLI** - Agent definitions, MCP server configuration
- **Gemini CLI** - Extension manifest, TOML agents (planned)
